# Module Expansion Template

This document outlines the pattern for expanding module pages with detailed content from MD files.

## Expansion Pattern

Each module page should include:

1. **Hero Section Updates:**
   - Update description with full overview from MD file
   - Keep existing badges and pricing

2. **Module Overview Section:**
   - Full overview paragraph from MD file
   - "Who This Module Is For" subsection with detailed list
   - Prerequisites note
   - Assessment Criteria with detailed rubric breakdown

3. **Learning Outcomes:**
   - Expanded list from MD file with detailed descriptions
   - Module Benefits subsection

4. **Weekly Breakdown:**
   - Update with detailed activities from MD file
   - Add keyConcepts array
   - Expand activities with full descriptions

5. **Assessment & Certification Section:**
   - Detailed rubric breakdown with percentages
   - Certificate details
   - ECTS credits

6. **Pricing & Enrollment Section:**
   - What's Included list
   - Enrollment Options

7. **Track Integration:**
   - Update with correct track name (AAC or AIO)
   - Add track integration note
