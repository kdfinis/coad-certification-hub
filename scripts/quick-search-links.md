## Download Instructions

1. **Click** a link above for the motive you need
2. **Browse** and select 8-10 high-quality images
3. **Download** images (use "Free Download" button)
4. **Rename** files: `{motive}_{source}_{id}.jpg`
   - Example: `executives_unsplash_abc123.jpg`
5. **Save** to: `public/images/{category}/{motive}/`
   - Example: `public/images/corporate-professionals/executives/`
6. **Run** update script: `npm run update-visual-db`

## Tips

- **Filter by**: Landscape orientation, High resolution
- **Choose**: Professional, modern, diverse images
- **Avoid**: Watermarked, low-quality, or copyrighted images
- **Verify**: All images are free for commercial use
