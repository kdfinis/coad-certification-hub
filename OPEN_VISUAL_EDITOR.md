# How to Open Visual Editor in Cursor

**Quick Steps to Open Visual Editor:**

## Method 1: Right-Click Menu
1. In Cursor, navigate to: `src/components/sections/HeroSection.tsx`
2. **Right-click** on the file in the explorer
3. Look for **"Open in Visual Editor"** or **"Open in AI Editor"**
4. Click it!

## Method 2: Command Palette
1. Press `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux)
2. Type: **"Visual Editor"** or **"AI Editor"**
3. Select the option to open visual editor
4. Choose `HeroSection.tsx` when prompted

## Method 3: File Tab
1. Open `src/components/sections/HeroSection.tsx`
2. Look for a **visual/design icon** in the file tab
3. Click it to switch to visual editing mode

## Method 4: Split View
1. Open `HeroSection.tsx` normally
2. Right-click the file tab
3. Select **"Split Editor"** or **"Open to the Side"**
4. In one pane, switch to visual editor mode
