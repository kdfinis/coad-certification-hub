## üîç Visual Editing Annotations Guide

### Annotation Types:

**`VISUAL-EDIT:`**
- Areas optimized for visual editing
- Can adjust styling, layout, positioning
- Example: `VISUAL-EDIT: Headline - Can adjust size, color, and gradient`

**`DRAG-DROP:`**
- Elements that can be dragged and dropped
- Can reorder or reposition
- Example: `DRAG-DROP: Buttons can be reordered or repositioned`

**`SPACING:`**
- Spacing values that can be adjusted visually
- Padding, margins, gaps
- Example: `SPACING: Container padding can be adjusted (pt-24 pb-16)`
